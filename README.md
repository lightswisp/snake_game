
# Snake Game

![](https://github.com/lightswisp/snake_game/blob/main/recording.gif)

## Run Locally

Clone the project

```bash
  git clone https://github.com/lightswisp/snake_game.git
```

Go to the project directory

```bash
  cd snake_game
```

Install dependencies

```bash
  bundle
```
or
```bash
  gem install gosu
```

Start the game

```bash
  ruby game.rb
```


## Controls

W, A, S, D

## Library used
https://github.com/gosu/gosu
